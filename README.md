## How to use the point tracker
- Install the following python modules:
  - xlsxwriter
  - openpyxl
- Fill out Data.txt with your student group
- Run point_tracker.py
- Upload the generated PointTracker.xlsx to Microsoft SharePoint and give access to your group
- Instructions on how to fill the table are in the .xlsx
